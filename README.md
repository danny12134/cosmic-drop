# 🌌 Cosmic Drop

O aplicație web care afișează zilnic o imagine sau video din arhiva NASA APOD
(Astronomy Picture of the Day).

## Funcționalități

- 🔭 Imaginea astronomică a zilei, direct de la NASA
- 📅 Selectează orice dată din arhivă
- ★ Salvează imagini favorite în browser
- 🎬 Suport video și imagini

## Cum se deschide

Deschide `index.html` în orice browser.
Nu necesită instalare sau server.

## Structura proiectului

```
cosmicdrop/
├── index.html   ← structura paginii
├── style.css    ← toate stilurile
├── script.js    ← logica aplicației (NASA API, favorite)
└── README.md    ← ești aici
```
